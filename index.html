Untuk Pengembang Aplikasi

Dokumentasi API TikSave

Integrasikan fungsionalitas pencarian video TikTok serta pemantauan kurs valuta asing real-time ke sistem backend Anda secara gratis.

1. API TikTok Downloader

GET https://albitiktok.vercel.app/?api=true&url={LINK_TIKTOK}

2. API Live Kurs Rupiah (IDR)

Gunakan parameter type=kurs untuk mengembalikan respons JSON berisi data konversi valas populer ke Rupiah beserta trend persentase harian.

GET https://albitiktok.vercel.app/?api=true&type=kurs

Query Parameters

Parameter Tipe Wajib Keterangan
api Boolean Ya Harus disetel ke true untuk mengaktifkan respons API berupa JSON.
url String Kondisional Tautan lengkap video TikTok. (Wajib jika menggunakan Downloader).
type String Tidak Gunakan kurs untuk mengambil data tabel nilai tukar Rupiah.
