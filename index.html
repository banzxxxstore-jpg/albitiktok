Status: AKTIF & GRATIS

1. Endpoint HTTP Request

GET https://albitiktok.vercel.app/?api=true&url={MEDIA_URL}

2. Query Parameters

Parameter Tipe Deskripsi
api string Wajib diisi dengan nilai true
url string URL valid (TikTok, YT, IG, FB) yang ingin dianalisis.

3. Contoh Kode Integrasi (Vanilla JS)

fetch('https://albitiktok.vercel.app/?api=true&url=https://vt.tiktok.com/ZS9PW3Bvm/')
  .then(res => res.json())
  .then(data => {
    console.log("Response:", data);
  })
  .catch(err => console.error(err));
